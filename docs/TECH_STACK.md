# Technology Stack & Conventions

## Core Frameworks

- **React**: Functional components with Hooks.
- **TypeScript**: Strict typing required. Avoid `any`.
- **Vite**: Build tool.

## UI / UX

- **Component Library**: `react-resizable-panels`.
- **Editor**: `@monaco-editor/react`.
- **Terminal**: `@xterm/xterm`.

## Key Libraries

- **@webcontainer/api**: For the Node.js runtime.
- **@sqlite.org/sqlite-wasm**: For persistent storage.
